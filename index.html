Preentrega 1 JavaScript

Prestamos ""El Mexicano""1

Simulador interactivo

Objetivos del proyecto:
Generar un script para ingresar a un servicio de prestamos con las siguientes caracteriaticas:
1.-Ingreso con correo y contraseña (3 oportunidades).
Al ingresar:
(a).-Mostrar un mensaje de bienvenida al ingresar.
(b).-Mostrar el menú de opciones
(c).-Se podra elegir una cantidad entre $5000.00 y $20000.00
(d).-Si el usuario no ingresa sus datos correctamente indicarle se comunique con soporte para restablecer la cuenta.
2.-El script calculará la cantidad a pagar por el prestamo que el usuario decida adquirir, sumando un 25% de intereses y un 16% de IVA.
3.-Al finalizar su compra el script mostrara un mensaje de agradecimiento.
